# Johannes January Career Track

Interactive six-month career development dashboard for Johannes January, an F-1 international graduate student targeting U.S. roles in Business Analysis, Data Analysis, Business Intelligence, Sales Analytics, Operations Analytics, and Project Analysis.

## What This Site Includes

- F-1 career command center
- Interactive progress dashboard
- Certification decision cards
- Printable certification brief pages
- Budget planner
- Monthly low-cost credential plan
- Portfolio project roadmap and detailed project briefs
- Week-by-week execution checklist with local browser progress tracking
- Role-fit matrix for target analyst roles
- Independent research track pages
- Job-search execution layer

## Local Use

Open `index.html` directly in a browser.

Progress checkboxes and certification tracking use `localStorage`, so progress is saved in the browser where the site is opened.

## GitHub Pages

This repository is designed to be hosted from the repository root with GitHub Pages.
